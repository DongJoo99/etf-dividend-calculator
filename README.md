# ETF 배당 계산기

투자금, 예상 연 배당률, 배당소득세, 투자 기간을 입력하면 예상 배당금과 기간별 총 수익을 보여주는 정적 웹사이트입니다.

## 파일 구성

- `index.html`: 화면 구조
- `style.css`: 디자인과 모바일 대응
- `script.js`: 계산 로직과 그래프

## GitHub 업로드

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 `index.html`, `style.css`, `script.js`, `README.md`를 업로드합니다.
3. 저장소 이름은 예를 들어 `etf-dividend-calculator`처럼 정하면 됩니다.

## Vercel 배포

1. Vercel에 로그인합니다.
2. `Add New Project`를 누릅니다.
3. GitHub 저장소를 선택합니다.
4. 별도 설정 없이 `Deploy`를 누릅니다.

배포가 끝나면 `https://프로젝트이름.vercel.app` 형태의 사이트 주소가 생성됩니다.
