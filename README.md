# ETF 배당 계산기

투자금, 예상 연 배당률, 배당소득세, 투자 기간을 입력하면 예상 배당금과 기간별 총 수익을 보여주는 정적 웹사이트입니다.

## 파일 구성

- `index.html`: 계산기 화면과 사용 가이드
- `style.css`: 디자인과 모바일 대응
- `script.js`: 계산 로직과 그래프
- `privacy.html`: 개인정보처리방침
- `disclaimer.html`: 투자 유의사항
- `contact.html`: 문의 안내
- `robots.txt`, `sitemap.xml`: 검색엔진 크롤링 보조 파일

## AdSense 준비 순서

1. https://adsense.google.com 에서 AdSense 계정을 만듭니다.
2. 사이트 주소로 `https://etf-dividend-calculator.vercel.app`를 추가합니다.
3. AdSense가 제공하는 사이트 확인 코드를 `index.html`의 `<head>` 안에 붙여넣습니다.
4. AdSense에서 제공하는 `ads.txt` 문구가 있으면 프로젝트 루트에 `ads.txt` 파일로 추가합니다.
5. 사이트 검토를 요청합니다.

승인 전에는 광고 코드를 임의로 넣지 않는 것이 좋습니다. 먼저 사이트 콘텐츠, 탐색 메뉴, 개인정보처리방침, 안내사항이 정상적으로 열리는지 확인하세요.
